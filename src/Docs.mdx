# Installation

```bash
npm install @mucsi96/ui-elements
```

# Usage

```ts
import '@mucsi96/ui-elements';
```

```css
body {
  background-color: var(--bt-background-color);
  color: var(--bt-text-color);
  font-family: var(--bt-font-family);
  margin: 0;
}
```
